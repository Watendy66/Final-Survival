# Final-Survival
离散数学归根结底还是数学，是需要做题的，而且最后突击是需要把知识点都给看一遍的才不会忘，虽然不难但琐碎的定义很多，所以要认真去记去背，还会有一些绕人的脑筋急转弯，并不能说明你的天赋，而只是思维是否打开是否灵活。
